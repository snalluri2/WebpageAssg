WEBSITE ASSIGNMENT
I am creating this website to practice the topics so far covered in my Web Systems Development class like how to make a directory using command line, how to convert the given directory into a git repository, how to push the converted repository into the remote server (GitHub Account).
I am using the following Linux, Vi and Git commands to make and commit the changes to the git based Web pages.

Purspose Of This Website :-
To understand and practice the command line instructions to create, commit and push the code for a GitHub based HTML website.

Linux Commands :-
PWD = Present Working Directory
CD = Current Directory
LS = List Files
Vi FILENAME (replace with the actual file) = To create and open an vi file
mkdir = Make Directory

Vi(Visual editor) Commands
I = Insert Mode (Esc = to come out of the insert mode)
: = Command Prompt for VI
W = Write Command
Q = Quit Command
WQ = Write and Quit

Git Commands
Git init = Create a RePo o
Git Add (Filename) = To add files to git for commiting
Git commit -m (messsage) = To commit the saved changes
Git Checkout -b (branch name) = To make a new branch and checkout
Git Checkout (Branch name) = To checkout branch
Git Merge (Branch Name) = To merge the branch
Git config = To cofigure git accordingly
Git clear -i = TO clear untracked branches
Git remote add origin (copy paste the link fo the the repo) = To add the remote server link
Git remote show = To see if there is an origin set
Git remote show origin = To see the link
